# Disk Space Full

## Overview

Available disk space has reached a critical level, preventing applications and operating systems from functioning normally.

---

## Symptoms

- Disk usage above 95%
- Write failures
- Log generation failure
- Database issues
- System instability

---

## Possible Causes

- Log accumulation
- Backup files
- Temporary files
- Large uploads
- Storage misconfiguration

---

## Verification Steps

1. Check disk utilization.
2. Identify large files.
3. Review log directories.
4. Verify backup retention.
5. Monitor storage health.

---

## Resolution Steps

1. Remove unnecessary files.
2. Archive old logs.
3. Expand storage capacity.
4. Clean temporary files.
5. Verify available space.

---

## Rollback Procedure

- Restore archived data if required.
- Validate application functionality.

---

## Monitoring

Monitor:

- Disk Usage
- Storage Growth
- File System Health

---

## Escalation

Severity: High

Escalate to:

- Infrastructure Team
- Storage Team
